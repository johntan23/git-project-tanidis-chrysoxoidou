Περιγραφή

Αυτή η ιστοσελίδα αποτελεί το project εξάσκησης στο Git για το IEK. Εδώ προσθέτουμε περιγραφή του έργου, τι αλλάξαμε και πώς δουλέψαμε με branches και commits.

Τι μπορείτε να αλλάξετε. Όλα θα τα βρείτε στα αρχεία του project.

  • Κείμενα και τίτλους σε αυτό το αρχείο (index.html)
  • Χρώματα/γραμματοσειρές στο assets/css/style.css
  • Μικρή συμπεριφορά στο assets/js/script.js
  • Νέα ενότητα/σελίδα π.χ. contact.html ή about.html
  • Προσθήκη εικόνων στον φάκελο assets/img/

Ένας τυχαίος τίτλος

  1. Εδώ θα μπορούσε να είναι ένα στοιχείο.
  2. Αλλάξτε αυτό το κείμενο.
  3. Θέλω commit και μήνυμα σε κάθε αλλαγή.
  4. Merge του branch στο main.
  5. Δημιουργία tag (π.χ. v1.0).

Λεπτομέρειες στο README.md.

Gallery – Εντολές Git

Εδώ το Μέλος 1 θα προσθέσει screenshots από το τερματικό με τις βασικές εντολές Git που εκτελέστηκαν. Κάθε εικόνα πρέπει να έχει περιγραφικό όνομα αρχείου (π.χ. git-status.png, git-log.png) και να αποθηκευτεί στον φάκελο assets/img/.

➤ Μπορείτε να αντικαταστήσετε ή να προσθέσετε δικές σας εικόνες μέσα στον φάκελο assets/img/. Για κάθε νέα εικόνα, δημιουργήστε και μία λεζάντα.
